ToMyPizza! is a comprehensive pizza order management system built as a frontend web application for a fictional pizza delivery business in Athens, Greece (Gravias Street). The application combines a customer self-service website with an internal order management system, featuring real-time tracking, analytics, and a fully responsive and modern design.

Technologies Used:

Frontend: HTML5, CSS3, Vanilla JavaScript

Frameworks/Libraries: Bootstrap 5, Chart.js, Font Awesome

APIs: Google Maps Embed API, Quotable API

Storage: Local Storage API for data persistence

Tools: VS Code, Git, Chrome DevTools
